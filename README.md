关于实验的详细信息可以查看[hoverwinter/HIT-OSLab](https://github.com/hoverwinter/HIT-OSLab)，实验环境的快速搭建参考[DeathKing/hit-oslab](https://github.com/DeathKing/hit-oslab)。
